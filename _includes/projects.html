Coding Projects

  Crypto Bot - (2020) Crypto bot is a secure cryptocurrency trade scheduler. Allows users to automatically purchase Bitcoin, Ethereum, and any other supported Coinbase Pro products at user-defined intervals. Effective for anyone looking for a tool to dollar-cost average into a position at low fees, and track their performance over time.

  JWT Auth Demo - (2020) JWT Auth Demo is a project built to prove-out token based API security. It includes a front-end and HTTP request/response interceptors to manage token lifecycle. Based on rest-api-tutorial by @makinhs.

  Concert Companion (SangItLive) - (2016) An Android app for live music lovers to help track upcoming tour dates, and past set lists for their favorite artists.
